# Система управления складом (Warehouse Manager)
Эта задача научит тебя инкапсуляции (защите данных) и взаимодействию между объектами.

Суть: Тебе нужно создать систему, которая следит за товарами на складе.

Что нужно сделать:

## Класс Product:
`Поля:`

| Тип данных | Название поля | Описание                    |
|------------|---------------|-----------------------------|
| String     | name          | Название товара             |
| int        | quantity      | Количество товара на складе |
| double     | price         | Цена за единицу товара      |
Добавь метод reduceQuantity(int amount), который уменьшает остаток.

## Класс Warehouse:
`Поля:`

| Тип данных     | Название поля | Описание                 |
|----------------|---------------|--------------------------|
| List<Product\> | inventory     | Список товаров на складе |
*Подсказка: используй `List<Product> inventory = new ArrayList<>()` и работай с ним как с динамическим массивом (через методы `.add()` и тд), подробнее можно найти в интернете.*

`Методы:`

| Название              | Описание                                                            |
|-----------------------|---------------------------------------------------------------------|
| addProduct(Product p) | Добавляет товар.                                                    |
| printInventory()      | Выводит на экран список всех товаров и их общуюстоимость на складе. |

*Твой челлендж: Сделай так, чтобы нельзя было создать товар с отрицательной ценой. Если кто-то пытается это сделать через конструктор или сеттер — выводи ошибку или ставь 0.*
