## Мини-шпаргалка по ArrayList (то самое «новенькое»)

Поскольку в задачах выше тебе нужно где-то хранить объекты, обычный массив `Product[]` может быть неудобен (у него фиксированный размер). Попробуй ArrayList.
- Создать: `ArrayList<Product> list = new ArrayList<>();`
- Добавить: `list.add(new Product(...));`
- Прочитать: `list.get(0);`
- Узнать размер: `list.size();`
