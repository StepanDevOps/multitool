# Система «Умный Дом» (SmartHome Hub)

> **Заказчик:** «У меня дома куча гаджетов: умные лампочки, термостаты и электронные замки. Мне надоело открывать пять разных приложений. Сделай мне единый пульт управления».

### Требования:

1. **Единый интерфейс:** Я хочу иметь список всех моих устройств и одной кнопкой «Выключить всё» переводить их в спящий режим.
2. **Разное поведение:**
   * **Лампочка** при включении должна позволять менять яркость (от 0 до 100%).
   * **Термостат** должен поддерживать заданную температуру. Не давай пользователю выставить `-100` или `+500` градусов — техника сгорит!
   * **Замок** можно только открыть или закрыть (никакой яркости или температуры там быть не должно).
3. **Отчетность:** Мне нужна функция «Статус системы», которая выведет в консоль текущее состояние каждого устройства (например: *«Лампа в гостиной: Вкл, Яркость 80%»*).
4. **Безопасность:** Состояние устройств (включено/выключено, градусы) не должно меняться «напрямую» извне. Только через специальные команды с проверкой условий.
